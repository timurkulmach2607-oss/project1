- Додано нові категорії для активів: "Мишка" та "Гарнітура".
- Випадаючий список "Категорія" винесено за межі форми для підтримки динамічного відображення інтерфейсу.
- Для категорії "Принтер" реалізовано появу додаткового поля "Підкатегорія", яка дозволяє вибрати:
  - Принтер (Пристрій)
  - Тонери (Toner)
  - Барабани (Drum unit)
  - Контейнери відходів (Waste toner container)
- Якщо вибрана підкатегорія для принтера, вона буде збережена в базу даних (наприклад, "Принтер - Тонери (Toner)").
